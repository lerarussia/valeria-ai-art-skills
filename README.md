# Valeria Titova — AI Art Skills

Practical skills for artists, curators and cultural practitioners working with generative systems, visual research and institutional applications.

These skills are not collections of generic prompts. Each one describes a repeatable workflow, defines quality criteria and helps an AI assistant make decisions within a specific artistic or professional context.

## Skills

### 1. Psycheia Visual Language

A project-specific visual direction system for creating images in the language of **Psyche / Psyche 3.0**: fragmented digital identity, bodily memory, post-internet mysticism, analogue traces, interfaces, reflections and machine error.

Use it to:

- develop scenes without losing the project’s visual identity;
- translate a theme into symbols, materials, palette and composition;
- avoid generic cyberpunk and commercial AI aesthetics;
- create production-ready prompts and evaluate results.

→ [Open the skill](skills/psycheia-visual-language/SKILL.md)

### 2. Curatorial Application Editor

A structured editor for open calls, grants, residencies, exhibitions, festivals and institutional proposals. It audits requirements, identifies missing material, develops artist and project texts and prepares a final submission checklist.

Use it to:

- analyse an opportunity before writing;
- adapt an existing portfolio or application;
- build an artist knowledge base for repeated submissions;
- prepare bios, statements, project descriptions, budgets and timelines;
- switch between fast-deadline, curatorial-depth and market-ready modes.

→ [Open the skill](skills/curatorial-application-editor/SKILL.md)

## Repository structure

```text
skills/
├── curatorial-application-editor/
│   └── SKILL.md
└── psycheia-visual-language/
    └── SKILL.md
```

Each skill is self-contained and uses a standard `SKILL.md` file with trigger metadata and operational instructions.

## How to use

Download the folder of the skill you need and add it to the skills directory of a compatible AI workspace. You can then invoke it by name or describe a task matching its purpose.

Example requests:

```text
Use Psycheia Visual Language to turn this scene into a prompt for a 3:4 installation visual.
```

```text
Use Curatorial Application Editor to audit this residency open call and prepare the missing texts.
```
