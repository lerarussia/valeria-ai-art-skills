# Artist Knowledge Base Builder

## Principles

Preserve evidence and uncertainty. Never merge similarly named projects, people, venues, or dates without support. Never turn an inference into a fact.

## Workflow

1. Inventory all sources with file name, date/version, language, and likely authority.
2. Extract atomic facts into these sections:
   - identity and contact;
   - positioning and themes;
   - education and training;
   - exhibitions, screenings, talks, residencies, awards;
   - projects and works;
   - commissions and collaborations;
   - teaching, writing, press, and links.
3. Normalize names, dates, locations, roles, media, dimensions, credits, and URLs while preserving original wording where it matters.
4. Reconcile conflicts using the newest authoritative source. Keep unresolved alternatives in a conflict log.
5. Deduplicate records without erasing meaningful distinctions such as group versus solo presentation.
6. Create reusable text layers: one-line descriptor, 50/100/200-word bios, full biography, practice statement, and project summaries. Draft only from verified facts and label newly written text.
7. Identify missing evidence, outdated claims, weak links, and facts requiring confirmation.

## Output

Return a human-readable knowledge base plus:

- source map;
- canonical records;
- reusable texts by length and language;
- conflict and uncertainty log;
- missing-material checklist;
- update date and version.

When updating an existing base, preserve stable identifiers and show what changed.
