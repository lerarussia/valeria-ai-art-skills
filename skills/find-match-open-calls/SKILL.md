# Open Call Finder & Matcher

## Workflow

1. Establish the artist profile: location, citizenship, career stage, media, themes, languages, travel ability, budget, access needs, goals, and exclusions. Use supplied materials; ask only for missing constraints that could change eligibility.
2. Search current primary sources. Treat aggregators as leads and verify every shortlisted opportunity on the official page.
3. Record deadline with timezone, eligible applicants, geography, fee, funding, required presence, duration, language, deliverables, rights, and application materials.
4. Exclude closed, unverifiable, ineligible, exploitative, or materially mismatched opportunities.
5. Score each remaining opportunity from 0–5 on:
   - eligibility certainty;
   - practice and theme fit;
   - career value;
   - financial feasibility;
   - production feasibility;
   - time-to-apply feasibility.
6. Explain evidence for every score. Do not inflate fit because an opportunity is prestigious.
7. Return a ranked table, then recommend `apply`, `consider`, or `skip`.

## Output

For every item include the official link, host, location/format, deadline and timezone, support offered, costs, fit score, eligibility status, strongest match, main risk, and next action. Separate confirmed facts from inference. Flag ambiguous rules instead of guessing.

If asked to continue, produce an application checklist and identify reusable versus opportunity-specific materials.
